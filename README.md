# <a name="microsoft-graph-training-module---build-aspnet-core-apps-with-microsoft-graph"></a>Microsoft Graph-Schulungsmodul – Erstellen von ASP.net-Kernanwendungen mit Microsoft Graph

In diesem Modul erfahren Sie, wie Sie mit dem Microsoft Graph .NET SDK zusammenarbeiten, um eine ASP.net-Kern-MVC-Webanwendung für den Zugriff auf Daten in Office 365 zu erstellen.

## <a name="lab---build-aspnet-core-apps-with-microsoft-graph"></a>Lab-Build ASP.net Core-apps mit Microsoft Graph

In dieser Übungseinheit erstellen Sie eine ASP.net-Kern-MVC-Anwendung, die mit Azure Active Directory (Azure AD) für die Authentifizierung & Autorisierung konfiguriert wird, die mithilfe des Microsoft Graph .NET SDK auf Daten in Office 365 zugreift.

- [ASP.net-Kern Microsoft Graph-Lernprogramm](https://docs.microsoft.com/graph/tutorials/aspnet-core)
